# Archive Vault

Permanent preservation area for external references, mirrored repositories, legacy firmware, datasheets, and technical fossils worth keeping.

## Purpose

This area is for reference material that should be preserved but not casually modified.

Use it for:

- mirrored repositories
- deprecated firmware
- legacy flight-controller notes
- datasheets
- PDF references
- abandoned tools
- historical documentation
- known-good snapshots

## Suggested Layout

```text
archive-vault/
├── mirrored-repos/
├── deprecated-firmware/
├── reference-pdfs/
├── datasheets/
├── old-flight-controllers/
├── utility-tools/
└── historical/
```

## Rule

Mirrors are fossils. Study them, index them, document them — but do not mutate them unless intentionally creating a derived project.

This is the engineering equivalent of putting weird old parts in labeled bins instead of a cursed coffee can.
