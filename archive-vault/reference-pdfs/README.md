# Reference PDFs

## Purpose

Storage area for important documentation:

- datasheets
- manuals
- technical references
- archived procedures
- protocol documentation
- avionics references

## Recommended Naming

Use readable filenames:

GOOD:

```text
stm32h743-reference-manual.pdf
esp32-p4-datasheet.pdf
mavlink-protocol-guide.pdf
```

BAD:

```text
manual2finalREAL.pdf
randomdownload.pdf
```

## Rule

If a document helped solve a problem once, preserve it.

Future-you will eventually need it at 2 AM.
