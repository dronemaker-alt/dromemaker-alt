# DJI Research

This directory collects DJI-specific hardware, firmware, teardown, and replacement-interface notes for the Drone Libre preservation effort.

## Files

- `mini2-teardown.md` — physical teardown and repair documentation template
- `firmware-notes.md` — firmware package, backup, and public tool references
- `hardware-mapping.md` — subsystem and connector mapping template
- `replacement-fc-concept.md` — early replacement or adapter flight-controller concept

## Documentation Rule

Separate notes into:

- observed facts
- measured data
- inferred conclusions
- speculative design ideas

That keeps the work useful instead of becoming a haunted junk drawer with Markdown support.
