## Inventory Rules

Each item should eventually be tagged as:

- `AVAILABLE`
- `INSTALLED`
- `DONOR`
- `DAMAGED`
- `TEST ONLY`
- `UNKNOWN`
- `NEEDS ORDERING`

Future updates should include quantity, condition, storage location, and assigned project.
