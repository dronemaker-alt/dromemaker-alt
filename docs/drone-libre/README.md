# Drone Libre

This directory contains the broader program documentation for the Drone Libre preservation, repair, modernization, and field-tool effort.

## Files

- `project-charter.md` — mission, scope, values, and non-goals
- `architecture.md` — system layers and interface philosophy
- `roadmap.md` — staged development path from documentation to prototype hardware

## Working Standard

The goal is to make the project understandable to another engineer, not just familiar to the person who built it.

Every design decision should eventually be backed by photos, measurements, test notes, or a clear reason.
