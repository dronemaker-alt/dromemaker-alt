# LilyGo P4 Notes

## Current Role

Primary candidate for portable handheld embedded diagnostics platform.

## Desired Functions

- portable diagnostics
- touchscreen UI
- firmware tools
- field reference system
- MAVLink utilities
- board diagnostics
- portable engineering workstation

## Advantages

- compact form factor
- integrated display
- ESP32 ecosystem compatibility
- strong experimentation platform

## Development Philosophy

Use larger development boards for architecture and debugging.

Use the handheld platform for:

- deployment testing
- UI refinement
- field ergonomics
- portable workflows

## Future Expansion

Potential integrations:

- CAN diagnostics
- UART adapters
- battery tools
- SD card logging
- OTA firmware workflows
- modular sensor attachments
